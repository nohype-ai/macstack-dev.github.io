Why use MacStack

Nothing new – except the ease of use

One Universal Command

mack update applies your whole stack config and updates everything

The Whole Stack

Explicate all aspects of the stack – far beyond software installs

Idempotent

MacStack updates only what you configure and preserves all else – down to individual entries in dotfiles and settings files.

All Configs are Optional

MacStack requires no configs at all. Expand your stack config at your own pace.

Continue Using Homebrew

MacStack employs Homebrew so you can continue using the quasi standard dependency manager.

Lightweight

MacStack is lightweight. No docker containers, virtual machines or MDM required.

Flexible Storage

Backup your stack config wherever you want, like in a git repo, cloud folder, memory card or external drive

On Any Mac

Blends into any Mac: fresh macOS installs, established machines, managed machines, headless machines

Use What You Know

MacStack brings together the configuration systems you already use. There's almost nothing new to learn.

Single Source of Truth

mack update transfers even JSON comments, so you can continue using comments as documentation.

Goodies Included

MacStack includes a few powerful customizations and helpers out of the box – no stack config required.

Where to Begin

Get started in one minute

curl -fsSL https://macstack.dev/setup.sh | zsh
1

Run above command

It will ensure Homebrew is present and set up MacStack

2

Set config path

Enter the folder path where your stack config will live

3

Apply your config

Verify that your stack update now works: mack update

What MacStack Covers

Go beyond installs and updates

Shell Customization

Prompt, environment, paths, functions, aliases

Git Configuration

Personal and general global git settings in ~/.gitconfig

Fonts

All fonts you would install, like Fira Code and SF Mono

CLI Tools

Like brew, git, zsh, mise, claude, gemini, uv, python, rust

App Store Apps

Apps and browser extensions from the Mac App Store

AI Agent Configurations

Settings + permissions of Cursor Agent, Gemini CLI, OpenCode

IDE Configurations

Settings + keyboard shortuts of VS Code based IDEs and Zed

VS Code Extensions

For any language support, agent integration and more

Git Repositories

All Git repos, their local paths, remotes, protocols
